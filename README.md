# GoobFuzz
GoobFuzz is a versatile web fuzzer designed for security testing and endpoint discovery.


# How to install

```txt
git clone https://github.com/glask1d/GoobFuzz
cd GoobFuzz
pip3 install -r requirements.txt
python3 GoobFuzz.py --help
```
